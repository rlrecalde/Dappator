Dappator.Sqlite NuGet Package 1.0.1.1 README
============================================

April 2024

This package supports .NET Framework 4.6.1 and above, .NET Standard 2.0 and above, .NET 5, .NET 6, .NET 7 and .NET 8.


Release Notes
=============

Bugfix regarding queries with Operators.In and Operators.Between


Dependencies:
============

- Dappator 1.0.1.1
- Dapper 2.1.44
- System.Data.SQLite 1.0.118 (for .NET Framework 4.6.1 and above)
- Microsoft.Data.Sqlite.Core 8.0.3 (for .NET Standard 2.0 and above, .NET 5, .NET 6, .NET 7 and .NET 8)
- SQLitePCLRaw.bundle_e_sqlite3 2.1.6 (for .NET Standard 2.0 and above, .NET 5, .NET 6, .NET 7 and .NET 8)
